Welcome to SoundTrackBox

This is a REACT Project

What is SoundTrackBox?
A small media player for music files. 
A user can create an account and start uploading a collection of songs. 
Those songs can be organized in playlists.
A user can play a song.
The player includes a canvas with an audio visualization.

How to deploy the project?
Start the backend -> SoundTrackBoxBackend
Use port 3001 for the backend
then run npm start in this project
